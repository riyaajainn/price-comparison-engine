# 🧬 Price Comparison & DNA Extraction Engine

A high-precision, production-grade price monitoring system.

## 🚀 Key Features
* **DNA Normalization**: Fixes storage typos (e.g., 5122GB) and removes marketing noise.
* **Stealth Scraping**: Uses Playwright Stealth to bypass Amazon bot blocks.
* **Waterfall Search**: Tries strict queries before broadening search.

## 🛠️ Tech Stack
* **Backend**: Node.js
* **Frontend**: Streamlit
* **Database**: MongoDB

## 🚦 Setup
1. Run 'npm install'
2. Start backend: 'node server.js'
3. Start dashboard: 'streamlit run dashboard.py'
